##### 简介
- 初次学习微信小程序练习项目
- 视频教程地址：`https://www.bilibili.com/video/av40455083/`
- 所用知识不多，主要是微信数据的请求与数据渲染.
- 视频播放页面，因为请求过来的数据中视频URL失效，替换了一个统一的视频URL.
